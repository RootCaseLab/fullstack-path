# fullstack-path

Этот репозиторий — старт моей долгосрочной программы развития от нуля до уровня fullstack-разработчика с практикой в сфере промышленного ПО.

Цели проекта:
- Освоить Python, веб-разработку (HTML/CSS/JavaScript) и современные фреймворки.
- Научиться проектировать и реализовывать системы класса ТОИР (управление техническим обслуживанием и ремонтами) и NCR/incident management.
- Разобраться в базовых практиках DevOps: работа с Git, CI/CD, контейнеризация, мониторинг.
- Исследовать и применять методы ИИ/ML для анализа данных инцидентов, предиктивного обслуживания и автоматизации расследований.

В репозитории по этапам фиксируются:
- учебные проекты и мини-серии задач;
- конспекты и шпаргалки;
- прототипы сервисов и инструментов, приближенных к реальным задачам бурения и промышленной эксплуатации.
